üÜò Need Help?

The button doesn't appear:

  • Refresh the page (press F5)
  • Check that you're on the KW CRM opportunities page
  • Make sure the extension is enabled in Chrome

Export fails or stops:

  • Check your internet connection
  • Make sure you're logged into KW CRM
  • Try refreshing and starting again

Can't find downloaded file:

  • Check your Downloads folder
  • Click the download icon in Chrome to see recent downloads
  • File name starts with: GiniExport_KW_Opportunities_
