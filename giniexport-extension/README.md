## ğŸ†˜ Troubleshooting

### The export button doesn't appear:
- Refresh the page (press F5)
- Make sure you're on the correct KW CRM opportunities page
- Check that the extension is enabled in your browser's extension settings

### Export stops or fails:
- Check your internet connection
- Make sure you're still logged into KW CRM
- Try refreshing the page and starting again
- Check if there's a popup blocker preventing the download

### Can't find the downloaded file:
- Check your browser's Downloads folder (usually in your home folder)
- Click the download arrow in your browser to see recent downloads
- The file name starts with "GiniExport_KW_Opportunities_"
